如何抢到 iPhone X 深空灰色 256G

发表于 2017-11-05 更新于 2019-08-06

吐槽

因为没有在首发的时候抢到 iPhone X，又想早点拿到 iPhone X，等3~4周是不可能的，所以就只能从官网预约到 Apple Store 购买。而官网11月3日没有开放预约，于是就只能在11月4号预约。

一大早八点钟起来预约 iPhone X，懒得起床就在手机上直接预约。iPhone X 的火爆程度不出我的意料，八点整还能刷出来有货，预约的时候经过手机短信、身份证、选店铺等繁琐的流程以后，iPhone X 果然没货了。然而在去年就抢过 iPhone 7 Plus 的我知道，苹果店每小时都会放出来一批货，于是我就顺手写了个脚本。

代码

GitHub Gist 地址：https://gist.github.com/ypwhs/b93086e748f8c8551521d67dcf3fa9f9

这代码思路很简单，就是循环查询是否有指定店铺的指定型号的 iPhone X 可供预约，如果有就利用 say 命令发出声音，然后只要手速够快，就能预约到 iPhone X，然后直接去店里取就行了。

店铺编号

店铺编号可以在这个地址查到：

https://reserve-prime.apple.com/CN/zh_CN/reserve/iPhoneX/stores.json

比如我想去西单大悦城买，就可以从刚才的地址中找到这一条：

从中我们可以知道店铺编号是 R388，如果我们还想添加别的店铺，就寻找对应的店铺编号，添加到 stores 里就好了。

设备型号

设备型号很容易查，只需要在官网上选择你的型号，添加到购物袋，就能从浏览器的地址栏看到型号，比如我选择 iPhone X 256G 深空灰色，就会看到这个地址：

https://www.apple.com/cn/shop/buy-iphone/iphone-x?product=MQA82CH/A&step=attach#

其中的 MQA82CH/A 就是这款 iPhone X 的型号，设置 product 为你想要的型号就可以了。

开始监控

代码都配置好以后，就可以开始监控 iPhone X 的状态了：

python check_iPhoneX.py

坐等有货

开始监控以后，只要目标店铺一有 iPhone X，就会语音提示对应店铺的名称，比如：西单，西单，西单。。。（Siri 声），如果等到有货的时候去官网抢，那一定是抢不到的，因为还得发短信，然后登录 iCloud 账户，填写身份证等。因此我们需要在 Apple Store 官方 APP 上面抢 iPhone X。

首先在 APP 里选择好自己想要的型号，然后在“这就是你的新 iPhone X”页面选择“预约，并到店购买”，通常第一次是抢不到的，因为虽然 APP 里不用验证手机号，但是还要输入身份证，以及选择预约时间。不过这里有一个小技巧就是，如果你在第一次喊出你想去的店铺名以后，你一定要先进去，然后填好身份证，不管有没有货，你都可以一直待在里面，然后等到下次再喊店铺名的时候，你就马上选择预约时间，点击预约，一般店铺会出来十秒左右，手快一定能抢到。

到店购买

约到了以后添加到 Wallet，然后去店里排队半个小时，刷爆了信用卡，买了两台 iPhone X。其实我是约到了三台的，首先约到了一台，然后妹子说她也要一台，我们俩就一起抢，在第二轮放货的时候两个人都抢到了，于是就尴尬地抢到了三台。不过由于周围的人都不需要 iPhone X，我也就没有购买这台预约到的 iPhone X 了。

店里的苹果员工强烈推荐了 AppleCare+，然后就买了。由于苹果的 AirPower 要到明年才出，我就顺手买了个 Belkin 无线充电板。

苹果这张图实在是太有杀伤力了，冲着这张图就想买齐整套设备！

使用体验

  • 首先最惊喜的一点是手机尺寸变小的同时，屏幕反而变大了，看起来很爽
  • 然后面容 ID 特别好用，除了在进行 Apple Pay 购买的时候需要按两下电源键比较麻烦以外，其他时候比之前用指纹要轻松省力得多
  • 还有无线充电非常方便，随手一放就可以充电，再也不用插拔接口了
  • iPhone X 在使用手势的时候有很多动画，用起来6得飞起，十分顺滑
  • 小刘海很容易接受，使用起来基本没有影响，有一些 APP 的 UI 还没有适配，顶部底部会留黑边比较难过
  • 有一个缺点是屏幕和玻璃面板都很容易成为指纹收集器，但是由于有防油渍涂层，不需要用水擦，只需要放在口袋晃一晃指纹就没了，还是比较方便的

总之，这是我用过的最好的 iPhone。
